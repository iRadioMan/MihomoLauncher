# MihomoLauncher

This utility is currently under development and may contain bugs and bad code. Use it at your own risk.
Feel free to open issues or submit pull requests.

## System requirements
- Windows 10 or newer 
- .NET Framework 4.8

## Usage
- Extract the files from the archive
- Copy your YAML configuration files to the "configs" directory
- Run MihomoLauncher.exe
- Select a core and click the "Download" button
- Select a configuration and click the "Start" button
- You can close the main window — the application will continue running in the system tray
